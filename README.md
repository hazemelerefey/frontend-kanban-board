## Overview

This project explores how a kanban-style board can be designed to make task movement, status tracking, and planning feel simple and responsive.

It focuses on interaction quality just as much as functionality.

---

## Product Use Case

Teams and individuals need boards that help them:
- move work across stages quickly
- keep priorities visible
- manage task details without friction
- maintain a clean overview of progress

---

## UI Highlights

- drag-and-drop task movement
- priority tagging system
- modal-based task editing
- clean visual grouping by status
- state-aware interaction using Redux Toolkit
